# 🔥 ULTIMATE FIGMA MAKE PROMPT (MAX COMPLETION + ULTIMATE UX + UI INSPIRED BY REFERENCE)

Navrhni **kompletní design + klikací prototyp** moderní **landing page (one-pager)** pro firmu **DOMIDOMI Energie**.

## Co musí být součást výstupu

* plně navržená **landing page**
* **design system** (tokens + typografie + grid + komponenty)
* **komponentový systém** (variants, states, responsive variants)
* **desktop + mobile + tablet**
* **klikací prototyp** (scroll, anchor nav, otevření formuláře, přepínání segmentů FO/FOP/PO, multi-step kalkulačka, sticky CTA)
* připravené pro **snadnou implementaci** (např. Next.js / Tailwind; konzistentní spacing, komponenty, auto-layout)

## Kontext značky (nesmí se ztratit)

**DOMIDOMI Energie** je odnož obchodní značky **DOMIDOMI**.
DOMIDOMI je partner pro **bezstarostnou domácnost** (marketplace / síť firem) pro bezstarostné a udržitelné bydlení: **fotovoltaika, tepelná čerpadla, energie, rekonstrukce domů a bytů, renovace (zateplení, okna, dveře), servis, dotační poradenství, financování…** a další. (ekosystém domidomi.cz)

DOMIDOMI Energie je **zprostředkovatel výhodných fixovaných cen energií** pro 3 segmenty:

1. **FO** (fyzické osoby)
2. **FOP** (podnikatelé)
3. **PO** (právnické osoby)

## Hlavní účel webu

Web musí být **prezentace + konverze v jednom** s **maximálním důrazem na konverzi** a “odvyprávěním příběhu” v souladu s brandem.
Primární konverze: **Nechat si připravit nabídku energií**.

## UI / Vizuální směr (INSPIRACE REFERENCÍ Z OBRÁZKU — detailní specifikace)

Vytvoř UI inspirované dodaným screenshotem (ber jako **vizuální referenci**, ne kopii). Styl musí být **moderní marketplace/partner landing page**, velmi přehledný, profesionální, důvěryhodný:

### Klíčové rysy vzhledu

* **Hero**: velký “image-led” blok s fotkou domu (nebo clean ilustrací domu) + tmavý gradient overlay pro čitelnost textu
* **Kontrastní primární akcent** (doporučeně červená / korálová jako “action color”), sekundární neutrály (šedá, bílá, antracit)
* **Zaoblené rohy** (12–16px), “soft shadow” karty, čisté hrany
* **Card-first layout**: sekce jsou skládány z karet a modulů (jako v referenci)
* **Sticky header** s CTA vpravo
* **Mikro-badge styl** (pill/badges) pro důvěru: “Zdarma”, “Nezávazně”, “Do 24 hodin”
* **Ikony**: jednoduché line / duotone, konzistentní tloušťka
* **Fotografie**: čisté, moderní, “home improvement / energy” vibe
* **Whitespace a rytmus**: velké mezery, jasná hierarchie, čitelná typografie

### Typografie (nastav v design systemu)

* Použij moderní sans (např. Inter / Manrope / Satoshi style).
* H1: 44–56 / semibold
* H2: 28–36 / semibold
* Body: 16–18 / regular
* Microcopy: 12–14 / regular/medium
* Důraz na čitelnost, krátké řádky, silná hierarchie.

### Barevné tokeny (definuj v design systemu)

* Primary (CTA): výrazný akcent (červená/korálová)
* Primary hover/active
* Secondary (tmavý background pro hero / sekce)
* Neutral 0–900 (bílá až antracit)
* Success / Warning (pro validaci a upozornění)
* Border color (jemná šedá)
* Shadow tokeny (2–3 úrovně)

### Tlačítka a form (konverzní styl)

* Primární CTA: výrazná, plná, velká, s ikonou šipky
* Sekundární CTA: outline nebo ghost
* Inputs: velké, jasné labely, helper text, error state, success state
* Zaoblení consistent s kartami

## CRO & “MAX COMPLETION RATE” UX pravidla (zásadní)

Navrhni stránku tak, aby maximalizovala dokončení formuláře a konverzi:

### 1) Minimal friction

* Preferuj **multi-step formulář** místo dlouhého (progressive disclosure)
* Každý krok max 2–3 pole
* Jasný progress (“Krok 1/3”) + uklidňující text “Zabere to ~2 min”
* Inline validace + chyby napsat lidsky

### 2) Důvěra v každém kroku

* U formuláře vždy: “Zdarma a nezávazně”, “Bez spamu”, “Ozveme se do 24 hodin”
* Trust badges a mini-statistiky (bez konkrétních čísel, pokud nejsou daná) — použij placeholdery, které lze snadno nahradit

### 3) Sticky CTA pattern (Stripe-style)

* Sticky bar při scrollu: “Získat nabídku” + zkrácený vstup (např. segment + energie)
* Na mobilu sticky bottom bar s CTA

### 4) Segmentace FO/FOP/PO

* Segment volba je **viditelná hned v heru** (tabs/chips)
* Přepnutí segmentu mění mikrocopy a benefity (ale layout zůstává konzistentní)

### 5) Redukce rozhodování

* Nabídni volby jako chips (Elektřina / Plyn / Obojí)
* Spotřeba volitelně: “Nevím spotřebu” → nabídnout alternativu (typ domácnosti / velikost / jistič)
